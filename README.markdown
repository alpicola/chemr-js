
chemr.user.js
=============

## Feature ##

Just provide incremental search index for reference sites by Greasemonkey.

Current Support:

 * http://search.cpan.org/
 * http://www2u.biglobe.ne.jp/~oz-07ams/prog/ecma262r3/
 * http://api.jquery.com/

## Key Binding ##

	C-L: focus input area
	RET: select first candinate
	C-U: clear input
